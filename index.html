Publicacion gratis

Subir a GitHub Pages paso por paso

  1. Crea una cuenta gratis en github.com.
  2. Crea un repositorio nuevo con el nombre que quieras, por ejemplo jrunlock-repairs.
  3. Sube estos archivos: index.html, styles.css, app.js, README.md y la carpeta assets.
  4. En GitHub entra a Settings, luego Pages.
  5. En Build and deployment elige Deploy from a branch.
  6. Selecciona la rama main y carpeta /root, luego guarda.
  7. Espera unos minutos y GitHub mostrara el enlace publico de tu pagina.
